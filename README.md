# CleanerKing

Windows system cleaner written in Java.

## Status

Experimental utility. Review the code and test in a disposable environment before using it on important machines.

## Setup

Install a compatible JDK, clone the repository, then build or run it with the project tooling present in the repository.

## Usage

Run only after reviewing what files or system locations the tool will touch.

## Configuration

Do not commit personal paths, tokens, credentials, or machine-local cleanup targets.
